# Node / npm 環境確認手順

本ドキュメントは、Viteプロジェクト操作前に
Node.js と npm が正しくインストールされているか確認する手順である。

---

## 1. Node バージョン確認

```bash
node -v
```

推奨：
```
v18 以上
（理想は v20 LTS）
```

---

## 2. npm バージョン確認

```bash
npm -v
```

npm 9 以上推奨。

---

## 3. npx 動作確認

```bash
npx -v
```

エラーが出なければOK。

---

## 4. Nodeが存在しない場合

### Mac (Homebrew)

```bash
brew install node
```

### または公式サイトからLTSをインストール

https://nodejs.org/

---

## 5. 推奨：Nodeバージョン固定（任意）

nvm を使う場合：

```bash
nvm install 20
nvm use 20
```

---

## 判定

以下が全てOKなら次へ進む：

- node -v が表示される
- npm -v が表示される
- npx がエラーにならない
