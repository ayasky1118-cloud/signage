# GitLab Runner 設定

CI/CD パイプラインが「保留中」のまま進まない場合の Runner 設定の確認ポイント。

---

## タグの一致

**タグ付き Runner は、同じタグを持つジョブのみを実行します。**

| 状況 | 結果 |
|------|------|
| Runner にタグ `signage_dev_runner` が付いている | タグを指定したジョブのみ実行 |
| ジョブに `tags: [signage_dev_runner]` を指定 | Runner がジョブをピックアップ |
| ジョブにタグを指定していない | Runner がジョブを無視 → **保留中** |

---

## 本プロジェクトの設定

- **プロジェクト Runner**: `signage_dev_runner`（タグ: `signage_dev_runner`）
- **`.gitlab-ci.yml`**: 各ジョブに `tags: [signage_dev_runner]` を指定

---

## 確認手順

1. **Settings** → **CI/CD** → **Runners**
2. プロジェクト Runner が **アクティブ**（緑）か確認
3. Runner のタグを確認
4. `.gitlab-ci.yml` の各ジョブに同じタグが指定されているか確認
