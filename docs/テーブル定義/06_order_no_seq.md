# order_no_seq（注文番号採番管理情報）

## テーブル概要

| 項目 | 内容 |
|------|------|
| 物理名 | order_no_seq |
| 論理名 | 注文番号採番管理情報 |

## 採番仕様

- 注文登録時、ログイン会社の当年データでレコードを取得する。
- 取得できた場合: 最終連番 + 1 で更新する。
- 取得できない場合: 連番 1 で新規作成する。
- 注文番号（order_no）形式: 年4桁 + last_number を0埋め6桁（例: 2025000001）
