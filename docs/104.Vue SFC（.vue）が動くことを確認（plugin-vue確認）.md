## 1. Vue（.vueファイル）を扱える状態か確認する（必須）

Viteは `.vue` をそのままでは解釈できないため、**Vue の SFC（.vue）を使う場合は `@vitejs/plugin-vue` が必須**である。

### @vitejs/plugin-vue が必要なケース

- **Vite プロジェクトが Vue テンプレート（create-vue / Vue 用スキャフォールド）で作成された場合**  
  → 通常は最初から含まれている
- **Vite プロジェクトが「Vanilla」や「React」等のテンプレートで作成された場合**  
  → `.vue` を追加で使いたい場合に、手動で `@vitejs/plugin-vue` を導入する必要がある

いずれの場合も、`vite.config.ts` の `plugins` に `vue()` を登録する必要がある。

### 1-1. 依存確認

frontend/package.json の devDependencies に `@vitejs/plugin-vue` があるか確認する。
