# GitHub vs GitLab 比較検討表

## 1. 料金プラン比較

| 項目 | GitHub | GitLab |
|------|--------|--------|
| **Free プラン** | $0/ユーザー/月 | $0/ユーザー/月 |
| **最低有料プラン** | Team: $4/ユーザー/月 | Premium: $29/ユーザー/月 |
| **年間費用（Free）** | 0円 | 0円 |
| **年間費用（有料・1人）** | 約7,200円 | 約52,200円 |

---

## 2. CI/CD 比較

| 項目 | GitHub | GitLab |
|------|--------|--------|
| **CI/CD ツール** | GitHub Actions | GitLab CI/CD |
| **Free プラン分数** | 2,000分/月 | 400分/月 |
| **設定ファイル** | `.github/workflows/*.yml` | `.gitlab-ci.yml` |
| **Self-hosted Runner** | 分数無制限・無料 | 分数無制限・無料 |
| **public リポジトリ** | 分数無制限 | 400分/月（同様） |

---

## 3. 主な機能比較

| 機能 | GitHub Free | GitLab Free |
|------|-------------|-------------|
| **リポジトリ** | 無制限（public/private） | 無制限 |
| **Issues** | ○ | ○ |
| **Projects** | ○ | ○ |
| **Pull Request / Merge Request** | ○ | ○ |
| **Wiki** | ○ | ○ |
| **パッケージストレージ** | 500MB | 10 GiB |
| **Dependabot** | ○ | ○（類似機能） |
| **Container Registry** | △（有料で拡張） | ○（10 GiB） |
| **Code Quality レポート** | △（Actions で実装） | ○ |
| **Container Scanning** | △（Actions で実装） | ○ |

---

## 4. 1人運用での使いやすさ

| 観点 | GitHub | GitLab |
|------|--------|--------|
| **学習コスト** | ◎ 低い | ○ やや高い |
| **日本語情報量** | ◎ 多い | ○ 少なめ |
| **マーケットプレイス / テンプレート** | ◎ 豊富 | ○ あり |
| **UI のシンプルさ** | ◎ シンプル | △ 多機能で複雑 |
| **Vue.js との相性** | ◎ 例が多い | ○ 問題なし |

---

## 5. 制限・注意点

| 項目 | GitHub Free | GitLab Free |
|------|-------------|-------------|
| **ユーザー数制限** | なし | グループあたり5人まで |
| **CI/CD 分数超過時** | 従量課金 | 従量課金（$10/1,000分） |
| **サポート** | コミュニティ | コミュニティ |
| **ストレージ** | 500MB（パッケージ） | 10 GiB |

---

## 6. 総合評価（1人・signage プロジェクト想定）

| 評価項目 | GitHub | GitLab |
|----------|--------|--------|
| **コスト** | ◎ 0円で十分 | ◎ 0円で十分 |
| **CI/CD 余裕** | ◎ 2,000分/月 | △ 400分/月 |
| **運用のしやすさ** | ◎ | ○ |
| **将来の拡張性** | ◎ | ◎ |
| **推奨度** | ◎ 推奨 | ○ 選択肢の一つ |

---

## 7. 選び方の目安

| こんな場合 | おすすめ |
|------------|----------|
| 1人で開発、まずはシンプルに始めたい | **GitHub** |
| CI/CD を多めに回したい | **GitHub**（分数が多い） |
| コンテナレジストリを最初から使いたい | **GitLab** |
| すでに GitLab を使っている | **GitLab** |
| 日本語の情報を重視したい | **GitHub** |
