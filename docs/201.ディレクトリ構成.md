# ディレクトリ構成

プロジェクト全体のフォルダ構成と、各フォルダの役割を説明する。

> **迷子用：** バックエンドだけを詳しく見たい場合は [202.backend_ディレクトリ構成.md](202.backend_ディレクトリ構成.md) を参照。

---

## 全体構成

```
signage/
├── frontend/       # フロントエンド（Vue 3 + Vite）
├── backend/        # バックエンド（FastAPI）
├── db/             # DBスキーマ・接続関連
├── docs/           # 手順書・仕様書
└── docker-compose.yml
```

---

## frontend/

Vue 3 + Vite + Vue Router + Tailwind CSS によるフロントエンドアプリケーション。

| パス | 説明 |
|------|------|
| `src/` | ソースコードのルート |
| `src/main.ts` | アプリのエントリポイント。Vue アプリのマウント、ルーター登録 |
| `src/App.vue` | ルートコンポーネント |
| `src/style.css` | Tailwind のエントリポイント |
| `src/pages/` | ページコンポーネント（ルートごとの画面） |
| `src/components/` | 再利用可能な UI コンポーネント |
| `src/composables/` | Composition API の再利用ロジック（API 呼び出しなど） |
| `src/router/` | Vue Router のルート定義 |
| `src/assets/` | 静的アセット（スタイル、画像） |
| `src/types/` | TypeScript の型定義 |
| `src/layouts/` | 共通レイアウト（ヘッダー・フッター等） |
| `index.html` | HTML のエントリポイント |
| `vite.config.ts` | Vite のビルド設定 |
| `package.json` | npm 依存関係 |

### 主要ファイル

| ファイル | 説明 |
|----------|------|
| `pages/Menu.vue` | 注文・看板管理メニュー画面 |
| `pages/OrderList.vue` | 注文一覧・検索画面 |
| `pages/OrderMain.vue` | 注文（新規・変更）画面 |
| `pages/OrderDetail.vue` | 看板編集画面 |
| `layouts/DefaultLayout.vue` | 共通レイアウト（ヘッダー・ログアウトボタン・ページ領域） |
| `components/AppHeader.vue` | 共通ヘッダー（DefaultLayout から利用） |
| `composables/useOrderApi.ts` | 注文一覧・注文1件取得（by-no）・注文新規登録 API 呼び出し |
| `assets/styles/common.css` | 共通スタイル（カラー変数、カードシャドウ等） |
| `assets/styles/order-list.css` | 注文一覧専用スタイル（テーブル、Flatpickr テーマ） |

## backend/

FastAPI + SQLAlchemy によるバックエンド API。

| パス | 説明 |
|------|------|
| `app/` | アプリケーション本体 |
| `app/main.py` | エントリポイント。FastAPI インスタンス、CORS、ルーター登録 |
| `app/core/` | アプリ全体の設定 |
| `app/db/` | DB 接続・セッション管理 |
| `app/routers/` | API エンドポイント（ルーター） |
| `app/models/` | SQLAlchemy ORM モデル |
| `.env` | 環境変数（DATABASE_URL 等） |
| `check_db.py` | DB 接続確認用スクリプト |
| `requirements.txt` | Python 依存関係 |

### 主要ファイル

| ファイル | 説明 |
|----------|------|
| `app/main.py` | FastAPI アプリ、CORS、`/health`、ルーター登録 |
| `app/core/config.py` | 設定クラス（DATABASE_URL 等を .env から読み込み） |
| `app/db/session.py` | SQLAlchemy Engine、SessionLocal、get_db |
| `app/routers/company.py` | `GET /companies`（会社一覧、DB 疎通確認用） |
| `app/routers/customer.py` | `GET /customers`（顧客一覧。注文フォームの顧客選択用。company_id で絞り込み） |
| `app/routers/order.py` | `GET /orders`（注文一覧検索）、`GET /orders/by-no`（注文番号で1件取得・order_item・納期・校正予定日含む）、`POST /orders`（新規登録・customerId 指定・納期・校正予定日は任意） |
| `app/models/company.py` | Company テーブルの ORM モデル |
