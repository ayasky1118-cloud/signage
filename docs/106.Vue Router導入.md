# Vue Router 導入手順（Vite + Vue3）

本ドキュメントは、Vue3 + Vite プロジェクトに Vue Router を導入する手順をまとめたものである。

---

## 1. パッケージインストール

```bash
npm install vue-router
```

---

## 2. ルーティングファイル作成

`src/router/index.ts` を作成。

```ts
import { createRouter, createWebHistory } from "vue-router"
import Menu from "../pages/Menu.vue"

const router = createRouter({
  history: createWebHistory(),
  routes: [
    { path: "/", component: Menu },
  ],
})

export default router
```

---

## 3. main.ts に適用

```ts
import router from "./router"

createApp(App).use(router).mount("#app")
```

---

## 4. App.vue を router-view のみにする

```vue
<template>
  <router-view />
</template>
```

---

## 5. 動作確認

ブラウザで以下の URL にアクセスし、Menu.vue（注文・看板管理メニュー）が表示されれば成功。

- **`/`** … `/menu` にリダイレクトされ、メニューが表示される
- **`/menu`** … 直接メニューが表示される

> 開発サーバーのポートは、5173 が使用中なら 5174 等になる。`npm run dev` の表示を確認すること。
