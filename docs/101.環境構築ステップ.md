## 必須ツールのインストール
- Docker Desktop（MySQLを動かすために必須）
- Node.js (v18以上)（Vueのフロントエンド用）
- Python (3.12)（バックエンド用）
- AWS CLI & AWS SAM CLI（Lambdaのローカル実行用）

### 手順 1-1：Node.js のインストール
1. フロントエンド（Vue 3）を動かすために必須
1. [Node.js 公式サイト](https://nodejs.org/ja)へアクセス
1. 最新版 LTSをダウンロードして実行
2. インストーラーの指示に従って完了

### 手順 1-2：Python 3.12（2026／03時点安定版）のインストール
1. [Python 公式サイト](https://www.python.org/downloads/)へアクセス
2. Python 3.12.x をダウンロード
3. 【重要】 インストーラーの最初の画面で 「Add Python to PATH」（Pythonをパスに追加する）というチェックボックスに必ずチェックを入れてから、インストールを開始

### 手順 1-3：Docker Desktop のインストール
1. [Docker Desktop 公式サイト](https://www.docker.com/products/docker-desktop/)にアクセス
2. 「Download Docker Desktop」 をクリックしてインストール
3. インストール後、Docker Desktopアプリを起動し、初期設定を完了

### 手順 1-4：AWS SAM CLI のインストール
1. [AWS公式のSAM CLIインストールガイド](https://docs.aws.amazon.com/ja_jp/serverless-application-model/latest/developerguide/install-sam-cli.html)から対象ファイルをダウンロード
2. インストーラーの指示に従って完了

### 手順 1-5：MapTiler API キーの取得（地図表示・住所検証用）

看板編集画面の地図表示および住所検証には MapTiler API キーが必要です。

1. [MapTiler Cloud](https://www.maptiler.com/cloud/) にアクセス
2. アカウントを作成またはログイン
3. ダッシュボードから API キーを発行
4. 以下の環境変数を設定する（詳細は [203.本番環境の環境変数.md](203.本番環境の環境変数.md) 参照）：
   - **バックエンド**: `backend/.env` に `MAPTILER_API_KEY` を記入（住所検証・ジオコーディング用）
   - **フロントエンド**: `frontend/.env` に `VITE_MAPTILER_API_KEY` を記入（地図タイル表示用）
5. `frontend/.env` は `frontend/.env.example` をコピーして作成する（Vite は `.env` を読み込む。`.env.example` を直接編集しても反映されない）
6. `.env` 作成・変更後はフロントエンド（Vite）を再起動する
7. 未設定時：住所検証はスキップされ、地図表示エリアには「VITE_MAPTILER_API_KEY を設定してください」と表示される
